# Tasks API

## Project Overview

This project is a simple REST API built using Node.js and Express. It allows users to create, view, update, and delete tasks. The API demonstrates backend concepts such as middleware execution, request validation, authentication using an API key, pagination, and structured error handling.

The API is intended for developers who want to connect a frontend application or test REST endpoints using tools such as Postman.

Core resources in this API include:

- tasks

Each task contains basic information such as a title, description, and completion status.
